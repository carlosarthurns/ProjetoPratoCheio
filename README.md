# 🍽️ Projeto Prato Cheio

Este sistema foi desenvolvido como uma atividade prática para a disciplina de **Laboratório de Programação 2**. O objetivo principal foi a fixação de conceitos fundamentais de **Programação Orientada a Objetos (POO)** em Java.

## 🎯 Objetivos do Projeto
* Implementar a lógica de um sistema de personalização de pratos.
* Aplicar pilares de POO como **Encapsulamento**, **Herança** e **Polimorfismo**.
* Exercitar a organização de pacotes e estrutura de código em Java.

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** Java
* **Paradigma:** Orientação a Objetos

## 📝 Contexto Acadêmico
O projeto faz parte do meu percurso de aprendizado em engenharia de software, onde explorei como transformar requisitos de um sistema real em código escalável e organizado.
